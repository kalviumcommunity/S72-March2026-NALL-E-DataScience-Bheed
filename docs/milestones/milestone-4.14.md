# Milestone 4.14 — Understanding Python Numeric and String Data Types

## Objective
To explore the specific data types within the BheedRadar tourism dataset and understand their roles in data science analysis.

## Key Learning Outcomes
1.  **Integers (`int`)**: Used for discrete counts (e.g., `visitor_count`, `is_holiday`). These columns are used for numerical comparisons and statistical modeling.
2.  **Floating Points (`float`)**: Used for continuous measurements (e.g., `temperature`). These columns are crucial for regression tasks.
3.  **Strings (`str`)**: Used for categorical labeling (e.g., `city`, `zone`). Strings are typically used for grouping and filtering data.

## Implementation: `notebooks/04_data_types.ipynb`
The notebook provides the following examples:
- Loading the dataset using Pandas.
- Accessing individual columns and verifying their types using the `type()` function.
- Demonstrating **type casting** (converting integers to floats).
